# 了解C#

## C#

C#是微软推出的一种基于. NET框架的、面向对象的高级编程语言。

 C#是一种由C和C++衍生出来的面向对象的编程语言。

 它在继承C和C++强大功能的同时去掉了一些它们的复杂特性，使其成为C语言家族中的一种高效强大的编程语言。
