# 了解Lua

## Lua

Lua是一个简洁、轻量、可扩展的脚本语言。

Lua有着相对简单的C API而很容易嵌入应用中。

很多应用程序使用Lua作为自己的嵌入式脚本语言，以此来实现可配置性、可扩展性。
